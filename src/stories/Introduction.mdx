# Portfolio Design System

이 Storybook은 **Joonho Kim Portfolio** 프로젝트의 UI를 문서화하고, 앱 전체를 띄우지 않고도 컴포넌트를 개발·검증하기 위한 공간입니다.

## 사이드바 구조

| 섹션 | 용도 |
|------|------|
| **Design System** | `globals.css` 색상 토큰, 타이포그래피 |
| **UI** | shadcn 기반 공통 컴포넌트 (`Button`, `Card`, `Dialog` …) |
| **Portfolio** | 포트폴리오·admin에서 쓰는 조합 컴포넌트 |

## 색상 / 테마를 바꾸려면

1. **정의**: `src/app/globals.css` — `:root` / `.dark`의 CSS 변수 (`--primary` 등)
2. **확인**: **Design System → Theme Colors** + 상단 툴바 **Theme** (light/dark)

Storybook은 색을 **정의하지 않고 보여주기만** 합니다.

## 스토리 추가 방법

컴포넌트 파일 옆에 `*.stories.tsx`를 두는 것을 권장합니다.

```tsx
// src/components/ui/card.stories.tsx
import type { Meta, StoryObj } from '@storybook/nextjs';
import { Card } from './card';

const meta = {
  title: 'UI/Card',
  component: Card,
  tags: ['autodocs'],
} satisfies Meta<typeof Card>;

export default meta;
type Story = StoryObj<typeof meta>;

export const Default: Story = {
  render: () => <Card className="p-4 w-64">Content</Card>,
};
```

## 명령어
